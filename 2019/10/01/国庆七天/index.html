vue和vue-cli是什么关系？

俗称 vue-cli 为脚手架，是一套大众化的前端自动化解决方案，他的核心是 webpack ，框架是vue，还有相关辅助插件组成。不怕浪费时间自己也可以按照自己的习惯搭一套，如果你有丰富的前端经验，可能构建一条合理的解决方案，不然会疏忽很多细节。grunt、gulp 时代，前端自动化规则，输入输出文件格式，文件的监听，文件的路径都是自己配置的。配置一套理想的方案也是相当耗费时间，所以 vue 就提供了一套面相大众的解决方案——vue-cli，他的输入输出规则，文件的打包路径，模块的命名基本上符合大众前端开发者的习惯，不过实际开发中需要一定的调整。

1.第一天，vue.js入门

1. v-for指令

和普通for一样，类似于wx:for

var app1 = new Vue({

绑定属性，使用data,v-bind简写为”:”

3. v-on指令

绑定事件

    <div id="app-3">
        <button v-on="{mouseenter:onEnter,mouseleave:onOut}" v-on:click="onClick">点击按钮</button>
        <form v-on:submit.prevent="onSubmit" v-on:keyup.enter="onEnter"><!--$event prevent stop阻止整页面刷新-->
            <input type="text">
            <button type="submit">提交</button>
        </form>
    </div>
var app3 = new Vue({
    el:'#app-3',
    methods:{
        onClick:function(){
            console.log("点击了");
        },
        onEnter:function(){
            console.log('mouse enter');
        },
        onOut:function(){
            console.log('mouse leave');
        },
        onSubmit:function(e){
            e.preventDefault();
            console.log('submit')
        },
        onKeyup:function(e){
            console.log('keyup')
        }
    }
})

4. v-model指令

  1. lazy控制鼠标点击外部后再实现双向绑定
  2. v-show控制是否显示和hidden类似

  4. trim去两端空格，在很多语言里面都有这个

        {{form}}

6.计算属性

把计算的过程存到计算属性中去，让它自己动态的计算。
它是一个function()里面可以写大量的逻辑。。。

<div id="app-5">
        <table border="1">
            <thead>
                <th>学科</th>
                <th>分数</th>
            </thead>
            <tbody>
                <tr>
                    <td>数学</td>
                    <td><input type="text" v-model.number="math"></td>
                </tr>
                <tr>
                    <td>英语</td>
                    <td><input type="text" v-model.number="english"></td>
                </tr>
                <tr>
                    <td>总分</td>
                    <td>{{sum}}</td>
                </tr>
                <tr>
                    <td>平均分</td>
                    <td>{{average}}</td>
                </tr>
            </tbody>
        </table>
    </div>
var app5 = new Vue({
    el:'#app-5',
    data:{
        math:90,
        english:110
    },
    computed:{
        sum:function(){
            return this.math + this.english;
        },
        average:function(){
            return (this.sum/2);
        }
    }
})

1.过滤器

    <div id="myapp">
        <div>
            <input v-model="length">mm
            <br>
            {{length | meter}}
        </div>
        <hr>
        <div>
            <input v-model="price">
            {{price | currency('USD')}}
        </div>
    </div>
Vue.filter('currency',function(val,unit){
    val = val || 0;
    unit = unit || '元';
    return val + unit;
});
Vue.filter('meter',function(val,unit){
    val = val || 0;
    unit = unit || '米';
    return (val/1000).toFixed(2) + unit;
});
new Vue({
    el:'#myapp',
    data:{
        price:10,
        length:10
    }
})

Vue.directive('pin',function(el,binding){
    var pinned = binding.value;
    var position = binding.modifiers;
    var warning = binding.arg;
    if(pinned){
        el.style.position = 'fixed';
        for(var key in position){
            if(position[key]){

1.混合mixins

混合重用一块相同的逻辑代码，例如显示或者隐藏

    <div id="app">
        <popup></popup>
        <tooltip></tooltip>
    </div>
var base = {
    methods:{
        show:function(){
            this.visible = true;
        },
        hide:function(){
            this.visible = false;
        }
    },
    data:function(){
        return{
            visible:false
        }
    }
}
Vue.component('tooltip',{
    template:`
    <div>
    <span @mouseenter="show" @mouseleave="hide">bys</span>
    <div v-if="visible">
        哈哈哈哈
    </div>
    </div>
    `,
    mixins:[base]
})
Vue.component('popup',{

        <button @click="show">Popup</button>
